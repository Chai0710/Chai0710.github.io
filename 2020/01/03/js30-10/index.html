★ 驚嘆號 ! 與雙驚嘆號 !!

驚嘆號 ! 是個邏輯運算符，名稱為 Logic NOT ，用在布林值上具有反轉 (Inverted) 的功能，雙驚嘆號 !! 就等於 反轉再反轉，等於轉回原本的布林值。

經過雙驚嘆號運算後，只會很單純出現 true 和 false 兩種。

在某些情況下，可以利用 !! 將變數轉型成 Boolean ，減少出錯機會。

例如希望0 和 null 被視為完全相同時：


  • 在 JavaScript 中，除了以下的值為 false 之外，其他都是 true

    • undefined, null
    • Boolean: false
    • Number: 0, NaN
    • String: ''

    ＊注意：空物件 {} 或是空陣列 []，也會視為 true！
